# @ianwalter/pino-print
> Makes Pino log lines easy to read

[![npm page][npmImage]][npmUrl]
[![CI][ciImage]][ciUrl]

![Screenshot](screenshot.png)

## Installation

```console
yarn add @ianwalter/pino-print
```

## CLI Usage

```console
node server.js | pino-print
```

## Options

* `--level, -l <debug|info>`
